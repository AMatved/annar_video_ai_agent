## 🔧 Технические детали

### Веб-поиск (бесплатный)
```python
# Использует DuckDuckGo - без API ключа!
from mcp_tools import web_search

result = web_search("TikTok trends 2025", max_results=10)
# → Возвращает реальные результаты из интернета
```

### Извлечение контента (бесплатный)
```python
# Использует BeautifulSoup - без API ключа!
from mcp_tools import extract_webpage

content = extract_webpage("https://example.com")
# → Извлекает текст с любой веб-страницы
```
