**Проблема:** Не успевает завершить за 5 итераций

**Решения:**

```python
# Вариант 2: Разбивать запросы на этапы
# Шаг 1: python agent.py "Найди тренды TikTok"
# Шаг 2: python agent.py "Создай идеи на основе найденных трендов"

# Вариант 3: Более конкретные запросы
# Вместо: "Найди тренды и создай идеи"
# Использовать: "Создай идею на основе трендов AI"
```

### 2. DuckDuckGo иногда возвращает ошибки
**Проблема:** Периодические ошибки соединения

**Решение:**
- Уже есть fallback на BeautifulSoup
- Можно добавить повторные попытки (retry)
