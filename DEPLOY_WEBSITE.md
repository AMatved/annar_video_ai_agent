# 🚀 Deploy Website - 30 Seconds

## Instructions:

### Step 1: Open Folder
Open in Explorer:
```
C:\Users\User\video_creator_agent\website
```

### Step 2: Go to Netlify
Open browser:
```
https://app.netlify.com/drop
```

### Step 3: Drag & Drop
Drag the `website` folder onto the Netlify page

### Step 4: Get URL
Netlify will give you a URL instantly like:
```
https://wonderful-site-123456.netlify.app
```

### Done! 🎉

You now have:
✅ GitHub repository: https://github.com/AMatved/annar_video_ai_agent
